# Álgebra

> **Curso de referencia del template.** Este curso NO es un curso activo del usuario — es un ejemplo completo de cómo debe verse un curso bien generado tras el setup inicial. Sirve como patrón visual cuando la IA genere nuevos cursos.

## Prerrequisitos externos
- Aritmética básica (operaciones con enteros, fracciones y decimales; jerarquía de operaciones).

Si el usuario no tiene este prerrequisito, el setup inicial debería generar un curso de aritmética antes de este.

## Objetivo del curso
Manejar con soltura expresiones, ecuaciones (lineales, cuadráticas y sistemas), factorización, funciones básicas e inecuaciones — nivel suficiente para arrancar trigonometría, cálculo u otros cursos integradores sin fricción algebraica.

## Estructura del temario
El árbol completo de temas está en [`temario.json`](./temario.json). Léelo para ver:
- Los 12 temas con sus `id`, prerrequisitos internos, dificultad y sesiones estimadas.
- Objetivos de aprendizaje y criterio de dominio por tema.
- Bibliografía asociada (Baldor + Sullivan).

**Resumen numérico:** 12 temas, ~34 sesiones estimadas (incluyendo `12-progresiones` como opcional), dificultad 1-4.

## Notas generales
- El tema `12-progresiones` está marcado como `opcional: true`. Es útil pero no es ruta crítica hacia cálculo, así que puede recortarse si el tiempo aprieta.
- El tema `11-problemas-de-planteo` es el más importante para transferencia real del conocimiento — priorizar aunque tome más sesiones de las estimadas.
- Baldor es la bibliografía principal por familiaridad hispanohablante; Sullivan aporta enfoque más moderno y de funciones/gráficas (temas 09-10).
