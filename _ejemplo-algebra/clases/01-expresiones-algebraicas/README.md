## Objetivo de esta sesión
Introducir el lenguaje algebraico: qué es una variable, coeficiente, constante, término y grado. Empezar a simplificar expresiones con términos semejantes.

## Recuperación activa inicial
Primera sesión del curso — no había tema anterior. Se preguntó al usuario qué asocia con "álgebra" para detectar preconceptos. Respondió "letras en vez de números y ecuaciones" — buen punto de partida, ya intuye la abstracción.

## Resumen / apuntes
- **Variable:** símbolo (típicamente x, y, z) que representa un valor desconocido o que puede cambiar.
- **Coeficiente:** el número que multiplica a la variable. En 3x² el coeficiente es 3.
- **Constante:** número sin variable. En 3x² + 5, el 5 es constante.
- **Término:** cada sumando de una expresión. En 3x² + 2x - 7 hay 3 términos.
- **Grado de un término:** suma de los exponentes de las variables. 3x²y tiene grado 3 (2+1).
- **Términos semejantes:** misma parte literal (mismas variables con mismos exponentes). Solo el coeficiente puede diferir.

**Regla clave para simplificar:** solo se pueden combinar términos semejantes. `3x + 5x = 8x`, pero `3x + 5y` no se simplifica.

## Ejercicios trabajados
1. Identificar variables, coeficientes y grado de: `5x³ - 2xy + 7` → resuelto correctamente.
2. Simplificar `3a + 5b - 2a + b` → usuario respondió `a + 6b`, correcto.
3. Simplificar `4x²y + 3xy² - x²y + 2xy²` → usuario primero mezcló x²y con xy² (no son semejantes). Se corrigió tras revisar la regla. Resultado: `3x²y + 5xy²`.
4. Traducir "el triple de un número aumentado en cinco" → usuario dudó entre `3(x+5)` y `3x+5`. Se discutió cómo la coma implícita cambia la traducción. Aclarado.
