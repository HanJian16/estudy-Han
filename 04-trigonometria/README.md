# Trigonometría

## Prerrequisitos externos
- 02-algebra (manejo de identidades, ecuaciones y funciones)
- 03-geometria (triángulos, circunferencia, ángulos)

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Dominar la trigonometría a nivel UNI: razones trigonométricas, identidades, reducción al primer cuadrante, ecuaciones y funciones trigonométricas, y resolución de triángulos. Es además herramienta directa para vectores y cinemática en Física.

## Notas generales
No arrancar este curso hasta tener avance sólido en Álgebra (identidades requieren fluidez operativa) y bases de Geometría. El usuario la usó implícitamente en Física universitaria, así que las razones básicas deberían reactivarse rápido; el reto suele estar en identidades y ecuaciones.
