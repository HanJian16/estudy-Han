# Álgebra

## Prerrequisitos externos
- 01-aritmetica (operaciones con fracciones, potencias y proporcionalidad como base operativa)

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Dominar el álgebra preuniversitaria a nivel UNI: teoría de exponentes, polinomios, factorización, ecuaciones e inecuaciones, funciones y logaritmos. Es el curso de mayor peso en la prueba de Matemática y base directa de Trigonometría y Física.

## Notas generales
El usuario ya llevó Cálculo universitario, así que conceptos como funciones no le son ajenos — el problema es la fluidez operativa (factorización, productos notables, manejo de exponentes), que se oxida con el desuso. Priorizar volumen de práctica operativa sobre re-explicación teórica.
