# Geometría

## Prerrequisitos externos
- 01-aritmetica (proporcionalidad, para semejanza y escalas)
- 02-algebra (solo lo básico: despeje de ecuaciones para problemas de cálculo de medidas)

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Dominar la geometría plana y del espacio a nivel del examen UNI: segmentos, ángulos, triángulos, congruencia y semejanza, circunferencia, áreas y sólidos geométricos, resolviendo problemas que exigen construcción y trazos auxiliares.

## Notas generales
La geometría UNI premia la visualización y los trazos auxiliares — el cuaderno físico es la herramienta principal. Es mayormente independiente de Álgebra, así que sirve para intercalar sesiones y variar cuando el usuario se sature de temas algebraicos.
