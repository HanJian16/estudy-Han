# Aptitud Académica y Humanidades

## Prerrequisitos externos
- Ninguno (el razonamiento matemático se beneficia de 01-aritmetica, pero puede avanzar en paralelo desde el inicio)

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Rendir bien la Prueba 1 del examen UNI: razonamiento matemático (series, conteo, situaciones lógicas), razonamiento verbal (comprensión lectora, analogías, oraciones incompletas) y humanidades básicas (lenguaje, historia del Perú, geografía, cívica).

## Estructura del temario
El árbol de temas de este curso está en `temario.json`. Léelo para ver:
- Lista completa de temas con `id`, prerrequisitos internos, dificultad y sesiones estimadas.
- Objetivos de aprendizaje y criterio de dominio por tema.
- Bibliografía asociada.

## Notas generales
Curso de "dosis pequeñas y constantes": mejor 2-3 sesiones cortas por semana intercaladas con los cursos fuertes que bloques largos. El razonamiento se entrena con práctica variada, no con teoría. Humanidades es principalmente repaso dirigido con material de academia — no profundizar más de lo que el examen exige.
