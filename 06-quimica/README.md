# Química

## Prerrequisitos externos
- 01-aritmetica (proporciones y regla de tres para estequiometría)

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Dominar la química preuniversitaria a nivel UNI: estructura atómica, tabla periódica, enlace químico, nomenclatura, reacciones y estequiometría, soluciones, y nociones de química orgánica, resolviendo problemas tipo examen.

## Estructura del temario
El árbol de temas de este curso está en `temario.json`. Léelo para ver:
- Lista completa de temas con `id`, prerrequisitos internos, dificultad y sesiones estimadas.
- Objetivos de aprendizaje y criterio de dominio por tema.
- Bibliografía asociada.

## Notas generales
Probablemente el curso más "de memoria" del plan y el que el usuario tiene más olvidado (solo lo vio en secundaria hace ~10 años). La repetición espaciada vía glosario.json es especialmente importante aquí (nomenclatura, valencias, tabla periódica). Puede avanzar en paralelo con los cursos de matemática porque casi no depende de ellos.
