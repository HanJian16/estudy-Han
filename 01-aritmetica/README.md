# Aritmética

## Prerrequisitos externos
- Ninguno

> Si este curso depende de otros, ábrelos primero cuando tengas dudas de base — no dupliques ese contenido aquí, solo enlázalo. Si no hay prerrequisitos externos, escribe "Ninguno".

## Objetivo del curso
Dominar la aritmética a nivel del examen de admisión UNI: numeración, divisibilidad, fracciones, razones y proporciones, porcentajes y mezclas, resolviendo problemas tipo examen con enfoque de razonamiento (no solo cálculo mecánico). Es la base numérica de Álgebra, Química y Aptitud Académica.

## Notas generales
El usuario egresó de secundaria hace ~10 años pero cursa ingeniería en la UNFV: hay memoria latente, así que las primeras sesiones deben diagnosticar rápido qué recuerda antes de re-explicar desde cero. En la UNI la aritmética cae mezclada con razonamiento matemático — priorizar problemas tipo examen sobre teoría extensa.
